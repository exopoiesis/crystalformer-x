# Structural Analysis Module

Post-generation analysis tools for evaluating crystal structures generated by CrystalFormer. All tools are **chemistry-agnostic** -- they work for any crystal system: ionic conductors, membranes, MOFs, zeolites, porous materials, etc.

## Installation

The analysis module requires `pymatgen` and `scipy`, which are included as dependencies:

```bash
pip install crystalformer-x
```

## Quick Start

### Python

```python
from pymatgen.core import Structure, Lattice
from crystalformer_x.analysis import VoronoiAnalyzer, PercolationAnalyzer

# Load or build a structure
structure = Structure.from_spacegroup(
    194, Lattice.hexagonal(3.16, 12.58),
    ["Mo", "S"], [[0, 0, 0.25], [0.333, 0.667, 0.621]]
)

# Void & layeredness analysis
va = VoronoiAnalyzer(r_probe=0.4)
vr = va.analyze(structure)
print(f"Max void radius:    {vr.max_void_radius:.2f} A")
print(f"Void fraction:      {vr.void_fraction:.2%}")
print(f"Layeredness score:  {vr.layeredness_score:.3f}")
print(f"Interlayer spacing: {vr.interlayer_spacing:.2f} A")

# Percolation / channel analysis
pa = PercolationAnalyzer(r_probe=0.4)
pr = pa.analyze(structure)
print(f"Percolation dim:    {pr.percolation_dimensionality}D")
print(f"Percolates a/b/c:   {pr.percolates_a}/{pr.percolates_b}/{pr.percolates_c}")
if pr.min_bottleneck is not None:
    print(f"Min bottleneck:     {pr.min_bottleneck:.2f} A")
```

### Command line

```bash
# Full analysis (Voronoi + percolation)
python -m crystalformer_x.analysis structures.csv --r-probe 0.4 --check-percolation

# Voronoi only (faster, skip percolation)
python -m crystalformer_x.analysis structures.csv --voronoi-only

# Custom output path and grid resolution
python -m crystalformer_x.analysis structures.csv -o results.csv --grid-resolution 0.15
```

The input CSV must have a `cif` column containing `Structure.as_dict()` string representations -- this is the format produced by `scripts/awl2struct.py`.

## Python API

### VoronoiAnalyzer

```python
class VoronoiAnalyzer(r_probe=0.4, grid_resolution=0.2)
```

**Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `r_probe` | float | 0.4 | Probe radius in A (0.4 ~ H+ ionic radius) |
| `grid_resolution` | float | 0.2 | Grid spacing in A. Smaller = more accurate, slower |

**Methods:**

- `analyze(structure) -> VoronoiResult` -- analyze a single `pymatgen.Structure`
- `analyze_batch(structures) -> list[VoronoiResult]` -- analyze a list of structures

**VoronoiResult fields:**

| Field | Type | Description |
|-------|------|-------------|
| `max_void_radius` | float | Radius of the largest empty sphere (A) |
| `void_fraction` | float | Fraction of cell volume accessible to the probe (0-1) |
| `layeredness_score` | float | Largest fractional gap along any lattice direction (0-1) |
| `interlayer_spacing` | float \| None | Gap between layers in A. `None` if < 2 atoms |
| `stacking_direction` | int | Direction with the largest gap (0=a, 1=b, 2=c) |

**Interpreting `layeredness_score`:**

| Score | Interpretation |
|-------|----------------|
| < 0.3 | Atoms distributed uniformly -- 3D framework |
| 0.3-0.6 | Moderate anisotropy -- may have preferred planes |
| > 0.7 | Clearly layered -- atoms clustered in well-separated sheets |

### PercolationAnalyzer

```python
class PercolationAnalyzer(r_probe=0.4, grid_resolution=0.3)
```

**Parameters:** Same as VoronoiAnalyzer.

**Methods:**

- `analyze(structure) -> PercolationResult` -- analyze a single structure

**PercolationResult fields:**

| Field | Type | Description |
|-------|------|-------------|
| `percolates_a` | bool | Continuous channel exists along lattice vector **a** |
| `percolates_b` | bool | Continuous channel exists along **b** |
| `percolates_c` | bool | Continuous channel exists along **c** |
| `percolation_dimensionality` | int | Number of percolating directions (0-3) |
| `min_bottleneck` | float \| None | Minimum clearance in the percolating region (A). `None` if no percolation |

**Interpreting `percolation_dimensionality`:**

| Dim | Meaning | Example |
|-----|---------|---------|
| 0 | No connected channels -- probe is trapped | Dense NaCl with large probe |
| 1 | 1D channel (tube) | Zeolite 1D channel |
| 2 | 2D channels (layered conductor) | Interlayer transport in MoS2 |
| 3 | 3D channels (framework conductor) | Open perovskite, MOF |

## Command-Line Interface

```
python -m crystalformer_x.analysis [-h] [-o OUTPUT] [--r-probe R_PROBE]
                                    [--grid-resolution GRID_RESOLUTION]
                                    [--voronoi-only] [--check-percolation]
                                    input
```

**Arguments:**

| Argument | Description |
|----------|-------------|
| `input` | Path to CSV with a `cif` column |
| `-o, --output` | Output CSV path (default: `<input>_analysis.csv`) |
| `--r-probe` | Probe radius in A (default: 0.4) |
| `--grid-resolution` | Grid spacing in A (default: 0.2) |
| `--voronoi-only` | Skip percolation (faster) |
| `--check-percolation` | Explicitly enable percolation (default unless `--voronoi-only`) |

**Output CSV columns:**

| Column | Source | Description |
|--------|--------|-------------|
| `row` | -- | Original row index |
| `n_atoms` | -- | Number of atoms in the structure |
| `max_void_radius` | Voronoi | Largest empty sphere radius (A) |
| `void_fraction` | Voronoi | Accessible volume fraction |
| `layeredness_score` | Voronoi | Gap-based layeredness (0-1) |
| `interlayer_spacing` | Voronoi | Layer separation (A) |
| `stacking_direction` | Voronoi | a, b, or c |
| `percolates_a` | Percolation | Channel along a (True/False) |
| `percolates_b` | Percolation | Channel along b |
| `percolates_c` | Percolation | Channel along c |
| `percolation_dim` | Percolation | Number of percolating directions |
| `min_bottleneck` | Percolation | Narrowest point in channels (A) |

## Algorithms

### Void analysis (grid-based)

1. **Supercell construction:** Atom positions are replicated in a 3x3x3 supercell to correctly handle periodic boundary conditions.
2. **KD-tree:** A `scipy.spatial.cKDTree` is built from the supercell positions for efficient nearest-neighbor queries.
3. **Grid sampling:** A regular 3D grid is created inside the unit cell with spacing `grid_resolution`. For each grid point, the distance to the nearest atom is queried from the KD-tree.
4. **Metrics:**
   - `max_void_radius` = maximum distance across all grid points
   - `void_fraction` = fraction of grid points where distance > `r_probe`

Computational cost: O(N_grid x log(27 x N_atoms)). For a typical 20 A cell with 0.2 A grid (100^3 = 10^6 points), analysis takes ~1 second.

### Layeredness (gap analysis)

1. For each lattice direction (a, b, c), project all atomic fractional coordinates onto that axis.
2. Sort the projected values and compute gaps between consecutive atoms, including the periodic wrap-around gap.
3. `layeredness_score` = maximum gap across all three directions.
4. `stacking_direction` = the direction with the largest gap.
5. `interlayer_spacing` = gap (fractional) x lattice parameter (A).

This method is fast (O(N log N) per direction), robust, and requires no external dependencies beyond numpy.

### Percolation (doubled-grid method)

1. Build the same distance grid as void analysis.
2. Mark grid points as **accessible** where distance > `r_probe`.
3. **Tile** the accessible grid 2x2x2 -> a (2N_a x 2N_b x 2N_c) array.
4. **Label** connected components with `scipy.ndimage.label` using 6-connectivity.
5. **Percolation test:** For each direction d, compare labels in the original cell with the shifted copy. If any grid point has the same label as its periodic image, the structure percolates in that direction.
6. `min_bottleneck` = minimum distance-to-nearest-atom among all grid points in the percolating components.

## Examples

### Screening generated structures for ionic conductors

```python
from crystalformer_x.analysis import VoronoiAnalyzer, PercolationAnalyzer

va = VoronoiAnalyzer(r_probe=0.4)   # H+ probe
pa = PercolationAnalyzer(r_probe=0.4)

candidates = []
for struct in generated_structures:
    vr = va.analyze(struct)
    pr = pa.analyze(struct)

    if (vr.layeredness_score > 0.7           # layered
        and pr.percolation_dimensionality >= 2  # 2D+ channels
        and pr.min_bottleneck > 0.5):           # wide enough
        candidates.append(struct)

print(f"{len(candidates)} / {len(generated_structures)} pass screening")
```

### Batch analysis from command line

```bash
# Analyze
python -m crystalformer_x.analysis output_194_struct.csv \
    --r-probe 0.4 --check-percolation -o screened.csv
```

### Comparing different probe sizes

```python
for r in [0.4, 0.8, 1.2, 1.6]:
    pa = PercolationAnalyzer(r_probe=r)
    pr = pa.analyze(structure)
    print(f"r={r:.1f} A: {pr.percolation_dimensionality}D percolation, "
          f"bottleneck={pr.min_bottleneck}")
```
